Caller Identifier

Platform:

ASP.NET Web Api, ReactJS, MS SQLServer

Description:

Caller Identifier is a web-based application designed to help users identify incoming calls, determine whether they are spam or authentic, and search for contact details based on the caller's name. The application aims to enhance user privacy and security by providing valuable information about unknown callers.
